Order of Operations:
Step 1:
int a = 5;
int b = 4;
int c = 2;
int d = a + b * c;
Console.WriteLine(d);

Step 2:
int a = 5;
int b = 4;
int c = 2;
int d = (a + b) * c;
Console.WriteLine(d);

Step 3:
int a = 5;
int b = 4;
int c = 2;
int d = (a + b) - 6 * c + (12 * 4) / 3 + 12;
Console.WriteLine(d);

Step 4:
int a = 5;
int b = 4;
int c = 2;
int d = (a + b) / c;
Console.WriteLine(d);

Precision and limits:
int a = 7;
int b = 4;
int c = 3;
int d = (a + b) / c;
int e = (a + b) % c;
Console.WriteLine($"quotient: {d}");
Console.WriteLine($"remainder: {e}");

Minimum and maximum limits:
int max = int.MaxValue;
int min = int.MinValue;
Console.WriteLine($"The range of integers is {min} to {max}");

Overflow condition:
int max = int.MaxValue;
int min = int.MinValue;
Console.WriteLine($"The range of integers is {min} to {max}");
int what = max + 3;
Console.WriteLine($"An example of overflow: {what}");

Rounding in double:

Range of Decimal type:
